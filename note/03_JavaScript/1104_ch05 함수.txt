01. 익명함수
	var 변수명 = function(매개변수){함수표현식}; : 호출은 변수명을 통해 호출. 
			       		         익명함수 정의 전에 사용불가

02. 화살표함수
	var 변수명 = (매개변수) => 함수표현식; : 매개변수가 1개일 경우 괄호생략. 
					    한줄짜리 return문은 return 생략

03. 선언적함수
	function 함수명(매개변수){} : 선언부가 제일 먼저 컴파일.
				선언적 함수 정의 위에서도 함수 호출 가능
				재정의시 선언적 함수를 먼저 정의하기 때문에 모든 함수호출 변형됨

04. 디폴트값을 갖는 매개변수
	function 함수명(매개변수=기본값){} : 매개변수 수를 적게 해도 기본값이 있어서 수행

05. 매개변수와 리턴값
	function 함수명(매개변수, 매개변수, 매개변수...){ 함수표현식};
		* 매개변수가 선언된 것과 같은 경우 : 수행
		* 매개변수가 선언된 것보다 많은 경우 : 뒷부분 무시
		* 매개변수가 선언된 것보다 적을 경우 : 전달되지 않은 매개변수는 undefinded

06. 가변인자함수 : 매개변수 갯수에 따라 변하는 함수(화살표함수는 불가)
	* Array(n) : 방의 갯수가 n개인 데이터가 빈 배열
	* Array(n,n,n,n) : [n,n,n,n]
	* Array() : 방의 갯수가 0인 배열 

09. 내부함수 : 함수내의 함수(함수 내에서만 사용)
	* 내부함수 사용 이유 : 충돌을 피하고자

10. 콜백함수 : 매개변수로 들어온 함수
	* 함수명(함수명);

11. setInterval : n밀리세컨마다 콜백함수 수행
	setInterval(() => {콜백함수}, n);
	clearInterval 사용시 setInterval에 변수할당 후 사용
		ex) setTimeout(() => clearInterval(id), n);
	
12. setTimeout : n밀리세컨후에 콜백함수 수행
	setTimeout(() => {콜백함수}, n);
	clearTimeout 사용시 setTimeout에 변수할당 후 사용
		ex) id = setTimeout(()=>{},n);
		     clearTimeout(id);

13. 타이머주의점 : 타이머를 제외하고 수행한 후 타이머 수행
