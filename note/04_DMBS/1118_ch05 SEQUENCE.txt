* SEQUENCE : 순차번호 생성기. 대부분 인위적인 PK 사용 용도

CREATE SEQUENCE 시퀀스네임
	* START WITH N : N부터 생성 (기본값1)
	* INCREASE BY N : N씩 증가 (기본값 1) / -N을 사용시 감소하며 내려감
	* MAXVALUE N : 최대값 (최대값 오버시 다시 1로 회귀)
	* MINVALUE N : 최소값
	* NOCYCLE : MAXVALUE를 넘어선 값이 다시 1로 시작하는 것을 방지
	* NOCACHE : 캐시메모리 사용 안함

	SELECT * FROM DUAL; 오라클이 제공하는 1행 1열짜리 테이블
	
	시퀀스명.NEXTVAL : 순차번호 생성
	시퀀스명.CURRVAL : 현재까지 진행된 순차번호
