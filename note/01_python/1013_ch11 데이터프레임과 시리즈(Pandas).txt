2절. 데이터프레임 만들기
	2.1 딕셔너리 리스트를 이용해서 데이터프레임 만들기
		df.to_dict : 데이터프레임을 딕셔너리로
		df.to_numpy() : 데이터프레임을 넘파이배열로
		df.dtypes : 열의 타입
		df.info() 

	2.2 리스트를 이용해서 데이터프레임 만들기
		df.head(n) : 상위 n개 행만 가져오기
		pd.options.display.max_columns : 열 display 갯수
		pd.options.display.max_rows : 행 display 갯수
		df.head(1).T : 열의 가독성이 떨어질때 전치행렬 적용

	2.3 read_csv
		pd.to_datetime : 문자형으로 읽어들인 날짜를 datetime형으로 변경
		'파일경로', skiprows=[n] : n번째 행을 제외하고 읽어오기		
		'파일경로', nrows=n : 상위 n행까지 읽어오기
		'파일경로', sep = ',', '|', '\t' : sep을 기준으로 읽어들이기

	2.4 패키지에서 가져오기
		iris 가져오기 방법 1 : sklearn (머신러닝 패키지)
		iris 가저오기 방법 2 : statsmodels (R 데이터)
		iris 가저오기 방법 3 : seaborn (시각화패키지)

3절. 이름(열,행) 지정하기
	3.1 열이름 지정하기
		.columns = ['열이름']

	3.2 행이름 지정하기
		.index = ['행이름']
		.index = range()
		데이터 조회 : .loc[행,열]
		특정열을 index로 변경 : .set_index(열이름)
		index를 컬럼으로 편입 : .reset_index()
			inplace=True : 할당하지 않고 바로 적용
		기존 index제거 후 새 숫자 index 초기화 : .reset_index(drop=True)
		datetime열에서 날짜 및 시간 추출 : .dt.year(month, weekdat...)

	3.3 레벨이름 지정하기
		.columns = [['레벨이름'],['열이름']]
		.columns.name = ['레벨이름']
		.index = [['레벨이름'],['행이름']]
		.index.name = ['레벨이름']

4절. 부분 데이터 조회
	4.1 단일 열 조회
		['열이름']
		.열이름

	4.2 loc을 이용한 조회
		df.loc[행이름, 열이름] : 행이름, 열이름으로 조회
			행이름과 열이름자리에 슬라이싱 from:to - from부터 to까지 (to 포함)
			행이름과 열이름자리에 리스트

	4.3 iloc을 이용한 조회
		df.iloc[행순번, 열순번] : 행순번, 열순번로 조회
			행순번과 열순번자리에 슬라이싱 from:to:by - from부터 to앞까지 by씩 (to 미포함)
			행순번과 열순번자리에 리스트

	4.4 조건으로 조회하기
		df[df['열이름']=='조회하고자 하는 데이터']
		df.loc[df['열이름']=='조회하고자 하는 데이터']
		df.loc[df['열이름']=='조회하고자 하는 데이터', 슬라이싱]
		df.loc[df['열이름'] 조건, 슬라이싱]
		조건연산자
			df[(df['열이름']조건) & df['열이름']조건)]
		결측치 확인
			data.info()
			data.isna().sum()

5절. 데이터 삭제 및 추가
