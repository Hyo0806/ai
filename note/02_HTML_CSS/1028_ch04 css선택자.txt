01. 태그선택자
	header, h1, li 등으로 태그를 이용하여 전체 적용

02. 전체선택자
	* : 전체선택자. 페이지 초기화 용도. 전체 페이지의 디자인을 초기화

03. id와 class선택자
	#id이름
	.class이름

04. 혼합선택자
	div#header : div중 아이디가 header인것
	div #header : div밑의 header
	class='클래스이름'
	class='클래스이름1 클래스이름2' : 클래스이름이 2개

05. 속성선택자
	input[type='password'] : input중 type이 password인것
	input[type^='pa'] : input중 type이 pa로 시작하는 것
	input[type$='rd'] : input중 type이 rd로 끝나는것
	input[alt] : input중 alt속성이 있는것 

06. 자손선택자, 후손선택자
	div p : 후손선택자. div 밑의 모든 p
	div > p : 자손선택자. div바로 밑의 p만

07. 동위선택자
	h3~div : h3 바로 다음에 오는 div동생들 다
	h3+div : h3 바로 다음에 오는 div동생 하나

08. 반응선택자
	:hover : 커서를 올리면 바뀌는것

09. 상태선택자
	input:focus : 클릭하면 색상이 바뀌는것
	input:disabled : 선택할 수 없는 메뉴
	input:enabled : 선택할 수 있는 메뉴
	input:read-only : 읽을수만 있는 메뉴(read-only, disabled)

10. 구조선택자
	:nth-child(n) : n번째 있는것
	:nth-child(2n) : 짝수번째
	:nth-child(2n+1) : 홀수번째
	:first-child : 첫번째
	:last-child : 마지막

11. 문자선택자
	::first-line : 문장의 첫번째 줄
	::selection : 마우스로 선택할때 보이는부분

12. 링크선택자
	::after : 링크가 걸린것 뒤에 추가하는것

13. 부정선택자
	:not(제외하고자하는것)
